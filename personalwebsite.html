Project Description

Training Center Website Project

This project is a personal web design assignment where I created a training center website using HTML and CSS. The main purpose of this project is to present course information in a clean and attractive way.

The website includes a header with navigation menu, a hero section with a catchy slogan, an enroll button, a course section and a footer with quick links.

Three courses are displayed in this project: Beginner Course, Intermediate Course and Advanced Course. Each course contains duration, fee and a short description. The design also uses smooth scrolling, card layout and responsive styling to make the website more user-friendly.
